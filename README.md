# Sorting Algorithms
Sorting Algorithms implemented in java with generic type

## Algorithms implemented

* Insertion Sort

* Merge-Sort (recursive)

* Bubble Sort

* Quick Sort (recursive)

## Complexity

* Merge-Sort (n log(n))

* Quick Sort (n log(n))

* Bubble Sort ( O(n²))

* Insertion Sort (O(n²))

## Time to sort(50000 integer values)

* Bubble Sort (15819 ms)

* Insertion Sort (1795 ms)

* Merge Sort (1436 ms)

* Quick Sort (151 ms)
